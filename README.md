## Clone Repository
git clone <repository-url>

## Install Dependencies
npm install

## Install Playwright
npx playwright install

# Run Tests

## Run All Tests
npx playwright test

## Run Specific Test
npx playwright test filename.spec.js

## Run in Headed Mode
npx playwright test --headed
